# Clasificación Automática de Imágenes OCT mediante Deep Learning

Proyecto universitario de Inteligencia Artificial enfocado en clasificación automática de imágenes OCT retinales utilizando ResNet50 y análisis experimental de hiperparámetros.

## Objetivo
Evaluar el desempeño de ResNet50 mediante diferentes configuraciones experimentales para clasificación de patologías retinales.

## Dataset
OCT2017 (Kermany et al.)

Clases:
- CNV
- DME
- DRUSEN
- NORMAL

## Experimentos realizados
- FT 20 capas
- FT 60 capas
- LR 1e-6
- LR 1e-4

## Tecnologías utilizadas
- TensorFlow
- Keras
- Scikit-learn
- Google Colab
